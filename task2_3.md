# MAP - отображение, карта

Элементы хранятся в виде набора пар <Ключ, Значение>, набор ключей - только уникальные значения

Отдельная структура для хранения пары Entry (два метода: getKey(), getValue())

Методы:
1. put(Ключ, Значение) - если такой ключ есть тогда меняем значение на новое, если нет - добавляем пару
2. get(Ключ) - получить значение по ключу, если ключ не найден - возвращаем null
3. getKeySet() - вернуть множество всех ключей (Set501)
4. getValues() - вернуть список всех значений (List501)
5. getEntries() - вернуть список всех пар (экземпляров типа Entry)
6. size()

Реализация:
Интерфейс Entry501
Интерфейс Map501

реализация на массивах
