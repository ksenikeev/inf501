# Музыкальная шкатулка

## Функции

1. Формирование, хранение, загрузка каталога JSON (номер трека, название, автор/исполнитель, путь к файлу)
2. Ввод нового трека (контроль наличия файла по указанному пути)
3. Удаление трека
4. Просмотр всех треков
5. Поиск трека по номеру, исполнителю, названию
6. Проигрывание трека
7. Остановка воспроизведения текущего трека

каталог
```json
{
  "traks": [
    {
      "name": "...",
      "author": "....",
      "file": "..."
    },
    {
      "name": "...",
      "author": "....",
      "file": "..."
    }
  ]
}
```
